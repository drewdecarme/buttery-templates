# @buttery/core/logger

A `@buttery/core` sub-path export that re-exports the `@buttery/logs` package.

## Purpose

Consolidates the dependency list in any of the `@buttery/tools` packages.

## Installation

```bash
yarn add @buttery/core
```

### Usage

```ts
import { ... } from "@buttery/core/logger"
```
